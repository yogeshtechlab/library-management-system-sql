# Library Management System SQL Project

## Overview

A Library Management System built using MySQL to manage books, members, librarians, and borrowing transactions.

## Features

* Manage library members
* Manage books inventory
* Track book borrowing and returns
* Generate reports using SQL queries
* Use foreign key relationships for data integrity

## Database Tables

* Members
* Books
* Librarians
* Borrow_Records

## Technologies Used

* MySQL
* Git
* GitHub

## SQL Concepts Used

* Joins
* Group By
* Aggregate Functions
* Foreign Keys
* Views
* Stored Procedures
* Triggers
